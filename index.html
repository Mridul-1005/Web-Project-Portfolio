Snake game project using C++

In this project, I have created a digital version of the timeless arcade game where players control a growing snake. The objective is simple yet challenging: guide the snake to consume food (usually represented by a fruit) while avoiding collisions with walls or the snake’s own tail. As the snake devours more food, it elongates, making the game progressively harder. The rules are straightforward: don’t hit the walls or bite your own tail. I have also implemented some key components such as initializing the game, rendering the game board, and updating the game state.

Asteroid Shooter Game using C++

In this captivating project, I have harnessed the power of C++ to recreate the timeless thrill of the iconic Asteroids arcade game. Imagine a black void—a cosmic canvas—where menacing asteroids drift ominously. My mission was to pilot a sleek spaceship through this celestial chaos, avoiding collisions while relentlessly firing torpedoes to shatter these rocky threats.

Racing Car game project using Java

In my third semester, I have created a racing car game using Java, in this game players select their dream can be it a sleek sports car, a rugged off-roader, or a futuristic hypercar. Each vehicle boasts unique handling characteristics, acceleration, and top speed. The track stretches before me, a winding ribbon of asphalt through scenic landscapes or urban sprawl and my mission is to set record lap times, beat opponents, and claim victory. I have fine-tuned the controls: arrow keys, WASD, or even a gamepad. The car responds to every nudge, drift, and hairpin turn.

Weather Forecast Android App using Java

In my fourth semester I have created a weather forecast app using Java, this app bridges the gap between users and meteorological data, offering real-time weather forecasts for specific locations. Users input their desired city or location. The app fetches weather data temperature, humidity, wind speed, and more for the specified area. I have harnessed external weather APIs (such as OpenWeatherMap) to retrieve data. The app communicates seamlessly with these services, pulling in real-time updates. The app identifies the city mentioned in the voice query. It validates the city name against a database of recognized locations.
